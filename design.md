# Joy@60th Invitation Website - Design Style Guide

## Design Philosophy

### Visual Language
**Elegant Sophistication with Floral Romance**: The design embodies timeless elegance through a sophisticated floral aesthetic that celebrates Joy's milestone 60th birthday with refined grace. Every element combines classical beauty with contemporary luxury, creating an atmosphere of celebration that feels both intimate and grand.

### Color Palette
**Primary Colors:**
- **Blush Rose (#F8E8E8)**: Soft, warm pink that forms the foundation of our palette
- **Dusty Pink (#E8C4C4)**: Muted elegance for secondary elements
- **Champagne Gold (#F4E4BC)**: Luxurious accent for highlights and special details
- **Sage Green (#A8C4A2)**: Natural complement for floral elements
- **Cream White (#FEFCF3)**: Pure, elegant base for text and backgrounds

**Supporting Tones:**
- **Mauve (#D4A5A5)**: Deeper pink for emphasis and depth
- **Soft Coral (#F2C2A2)**: Warm accent for interactive elements
- **Pearl Gray (#F5F5F5)**: Subtle neutral for text and borders

### Typography
**Display Font**: Playfair Display (Serif)
- Used for headings, event title, and elegant emphasis
- Conveys sophistication and timeless beauty
- Large sizes for maximum impact

**Body Font**: Inter (Sans-serif)
- Clean, modern readability for all content
- Excellent legibility across devices
- Professional yet warm character

**Accent Font**: Dancing Script (Script)
- For decorative elements and personal touches
- Adds romantic, handwritten elegance
- Used sparingly for special emphasis

## Visual Effects & Styling

### Background Design
**Continuous Floral Elegance**: A seamless, subtle floral pattern in soft pink tones creates a cohesive background throughout the entire website. The pattern features delicate rose petals and botanical elements in very low opacity (5-10%), providing texture without overwhelming content.

### Hero Section Treatment
**Cinematic Floral Backdrop**: The hero section features a stunning generated image of an elegant floral arrangement or garden scene with soft pink roses and peonies. The image is treated with a subtle overlay to ensure text readability while maintaining the natural beauty of the florals.

### Interactive Elements
**Sophisticated Hover States**: All interactive elements feature elegant transitions with soft pink glows and subtle scale transforms. Buttons use a gentle lift effect with shadow expansion, while cards tilt slightly with enhanced floral borders.

### Animation Library Integration
**Anime.js**: Smooth, elegant transitions for form steps, countdown numbers, and page reveals
**Typed.js**: Elegant typewriter effect for the main heading "Joy@60th" and key event details
**Splide**: Refined carousel for photo gallery with custom pink navigation dots
**Splitting.js**: Letter-by-letter animations for decorative text elements

## Layout & Composition

### Grid System
**Symmetrical Elegance**: A balanced 12-column grid system ensures harmonious proportions throughout the site. Content is centered with generous white space, creating a sense of luxury and refinement.

### Section Organization
**Flowing Narrative**: Each section transitions seamlessly into the next through subtle visual cues and consistent floral motifs. The layout guides visitors naturally from discovery to RSVP completion.

### Card Design
**Elevated Elegance**: All content cards feature soft shadows, rounded corners, and subtle floral border accents. The design creates depth while maintaining the sophisticated aesthetic.

## Decorative Elements

### Floral Motifs
**Subtle Botanical Accents**: Delicate rose illustrations, petal scatter effects, and vine borders appear throughout the design. These elements enhance the theme without overwhelming the content.

### Iconography
**Elegant Line Icons**: Custom floral-themed icons for navigation, features, and interactive elements. Icons use the primary color palette with consistent stroke weights.

### Texture & Depth
**Layered Sophistication**: Subtle textures, gentle gradients, and soft shadows create visual depth while maintaining the clean, elegant aesthetic.

## Responsive Design Principles

### Mobile Elegance
**Touch-Friendly Luxury**: All interactive elements are optimized for touch with generous tap targets. The floral design scales beautifully across all screen sizes.

### Tablet Refinement
**Balanced Composition**: The layout adapts to medium screens while maintaining visual hierarchy and elegant proportions.

### Desktop Grandeur
**Full Feature Display**: The complete floral aesthetic is showcased on larger screens with enhanced animations and detailed decorative elements.

## Accessibility & Performance

### Color Contrast
**WCAG Compliance**: All text maintains a minimum 4.5:1 contrast ratio against backgrounds, ensuring readability while preserving the elegant aesthetic.

### Animation Considerations
**Reduced Motion Support**: All animations respect user preferences for reduced motion while maintaining visual elegance through static design elements.

### Performance Optimization
**Efficient Loading**: Images are optimized for web delivery, and animations are performance-conscious to ensure smooth interactions across all devices.

## Brand Consistency

### Visual Hierarchy
**Clear Information Architecture**: Typography scale, color usage, and spacing create a clear hierarchy that guides users through the invitation experience.

### Emotional Tone
**Celebratory Elegance**: Every design decision reinforces the joy and sophistication of this milestone celebration, creating an emotional connection with guests.

### Memorable Impact
**Lasting Impression**: The cohesive floral design creates a memorable experience that reflects the significance of Joy's 60th birthday celebration.
